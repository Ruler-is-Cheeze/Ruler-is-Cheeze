## 編集方法（みんあゆ向け）
1. index.html等をクリックして、右上に出てくる鉛筆みたいなマーク（編集ボタン）を押す。
2. コードを適宜編集する
3. 右上の「Commit Changes」という緑色のボタンを押す。
4. 編集は即座に反映されて、ちょっと待つとWebサイトに変更が反映される。（更新がされなかったら、一回ブラウザのキャッシュクリアをすると良い）
